# Shared Exposure Pattern Analysis

## MSSP Use Case
Identify recurring exposure patterns across customers.

## MCP Context
- Exposure types
- Technologies
- Misconfigurations

## Prompt
You are analyzing exposure trends across your MSSP portfolio.

Using ERM MCP data:
- Identify exposure patterns appearing across multiple customers
- Group by technology or misconfiguration type
- Highlight opportunities for standardized remediation

Provide:
1. Common exposure patterns
2. Affected customers
3. Recommended portfolio-wide fixes

## Expected Output
- Pattern-based analysis
- Efficiency-focused recommendations
