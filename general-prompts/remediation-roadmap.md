# Remediation Roadmap

## Use Case
Create a prioritized remediation plan based on ERM findings.

## MCP Context
- Exposure severity
- Asset criticality
- Threat intelligence
- Existing controls

## Prompt
You are developing a remediation roadmap using ERM MCP insights.

Based on current exposures:
- Prioritize remediation actions
- Align fixes to risk reduction
- Identify quick wins vs long-term improvements

Provide:
1. Prioritized remediation list
2. Estimated effort vs impact
3. Strategic recommendations

## Expected Output
- Clear prioritization logic
- Practical remediation guidance
