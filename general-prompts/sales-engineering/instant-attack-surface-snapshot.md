# Instant Attack Surface Snapshot

## Demo Goal
Provide an immediate, high-level view of the customer’s external attack surface.

## MCP Context
- Domains
- IPs
- Cloud assets
- Internet-facing services

## Prompt
You are assisting in a live customer demo using Check Point Infinity ERM.

Using MCP data:
- Summarize the customer’s external attack surface in plain language
- Highlight the number and types of exposed assets
- Call out anything that stands out as unexpected or risky

Provide:
1. A concise attack surface overview
2. 2–3 notable observations
3. Why this matters from a risk perspective

## Expected Output
- Fast, conversational tone
- Executive-friendly language
- Immediate “aha” moments
