# Renewal Readiness Assessment

## CSM Use Case
Gauge renewal risk and prepare mitigation actions.

## MCP Context
- Value delivered
- Risk trends
- Engagement indicators

## Prompt
You are assessing renewal readiness for a customer.

Using ERM MCP data:
- Evaluate perceived value and outcomes
- Identify potential renewal risks
- Recommend actions to strengthen renewal position

Provide:
1. Renewal readiness assessment
2. Risk indicators
3. Recommended engagement actions

## Expected Output
- Internal CSM-facing tone
- Action-oriented insights
