# Customer Remediation Progress Tracking

## MSSP Use Case
Track remediation effectiveness over time.

## MCP Context
- Exposure history
- Closure timestamps
- Severity trends

## Prompt
You are tracking remediation progress for a managed customer.

Using ERM MCP data:
- Identify resolved vs outstanding exposures
- Measure risk reduction over time
- Highlight stalled remediation efforts

Provide:
1. Progress summary
2. Remaining high-risk items
3. Recommended follow-ups

## Expected Output
- Metrics-oriented reporting
- Clear accountability
