# Adoption & Usage Review

## CSM Use Case
Assess ERM adoption and identify opportunities to increase usage.

## MCP Context
- Feature usage indicators
- Alert interaction
- Asset coverage

## Prompt
You are reviewing ERM adoption for a customer.

Using MCP data:
- Assess how actively ERM is being used
- Identify underutilized capabilities
- Recommend ways to improve adoption

Provide:
1. Adoption assessment
2. Gaps or missed opportunities
3. Enablement recommendations

## Expected Output
- Supportive, non-judgmental tone
- Enablement-focused guidance
