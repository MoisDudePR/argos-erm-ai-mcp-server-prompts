# New Customer Onboarding Snapshot

## MSSP Use Case
Quickly establish baseline visibility for a new customer.

## MCP Context
- Asset discovery
- Initial exposures
- Technology stack

## Prompt
You are onboarding a new customer into your MSSP ERM service.

Using ERM MCP data:
- Identify all externally exposed assets
- Highlight immediate high-risk exposures
- Establish a baseline risk posture

Provide:
1. Initial exposure snapshot
2. Immediate risks
3. Recommended first actions

## Expected Output
- Fast, impactful insights
- Onboarding-friendly tone
