# SLA Breach Detection

## MSSP Use Case
Identify customers approaching or exceeding SLA thresholds.

## MCP Context
- Exposure age
- Severity
- Alert timestamps
- Remediation status

## Prompt
You are monitoring SLA compliance across multiple customers.

Using ERM MCP data:
- Identify exposures exceeding SLA thresholds
- Group by customer and severity
- Highlight recurring SLA issues

Provide:
1. SLA breaches by customer
2. Severity and duration
3. Recommended escalation actions

## Expected Output
- Operations-focused language
- Clear escalation indicators
