# Before vs After ERM Value Story

## Demo Goal
Demonstrate the value of ERM compared to not having visibility.

## MCP Context
- Asset discovery
- Exposure detection
- Risk prioritization

## Prompt
You are explaining ERM value to a prospective customer.

Using ERM MCP data:
- Describe what visibility looks like without ERM
- Contrast it with what ERM reveals
- Emphasize risk reduction and efficiency gains

Provide:
1. “Before ERM” view
2. “After ERM” view
3. Key value takeaways

## Expected Output
- Clear contrast
- Sales-friendly language
