# Check Point ERM MCP Prompt Library
A curated collection of high-quality AI chat prompts designed for use with the **Check Point Infinity External Risk Management (ERM) MCP Server**.

This repository provides reusable, battle-tested prompts to help security teams:
- Explore and understand their external attack surface
- Analyze exposure and risk using ERM data
- Investigate alerts and threat activity
- Generate executive and technical reports
- Accelerate investigations using AI-assisted workflows

## Who This Is For
- Security analysts and threat researchers
- Sales & Sales Engineering teams
- MSSPs and SOC teams
- Anyone leveraging ERM MCP with AI assistants

## How to Use
1. Connect your AI assistant to the ERM MCP Server
2. Select a prompt from the relevant category
3. Paste or adapt the prompt into your chat session
4. Refine results iteratively using MCP context
