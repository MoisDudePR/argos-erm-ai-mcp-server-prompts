# Attacker Perspective View

## Demo Goal
Reframe ERM findings from an attacker’s point of view.

## MCP Context
- Exposed assets
- Services
- Vulnerabilities

## Prompt
You are an external attacker performing reconnaissance on this organization.

Using ERM MCP data:
- Identify what would be visible to an attacker
- Highlight the most attractive targets
- Explain how an attacker might prioritize targets

Provide:
1. Attacker’s-eye view summary
2. Top 3 targets of interest
3. Defensive implications

## Expected Output
- Story-driven explanation
- Easy for non-technical audiences to follow
