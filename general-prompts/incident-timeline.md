# Incident Investigation Timeline

## Use Case
Reconstruct a potential external security incident.

## MCP Context
- Exposure changes
- Alerts
- Threat activity
- Asset history

## Prompt
You are investigating a suspected external security incident using ERM MCP data.

Reconstruct:
- When exposure first appeared
- Subsequent changes or escalations
- Relevant alerts or threat activity

Provide:
1. Timeline of events
2. Key indicators of compromise or risk
3. Suggested containment actions

## Expected Output
- Chronological structure
- Clear investigative logic
