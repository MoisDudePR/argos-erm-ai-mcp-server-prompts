# Customer Alert Triage Queue

## MSSP Use Case
Support centralized SOC triage across tenants.

## MCP Context
- Alerts
- Customer mapping
- Asset criticality
- Threat intelligence

## Prompt
You are operating a centralized MSSP SOC.

Using ERM MCP data:
- Triage alerts across all customers
- Prioritize alerts by risk and customer impact
- Identify alerts requiring immediate action

Provide:
1. Prioritized alert queue
2. Customer impact summary
3. Next investigative steps

## Expected Output
- SOC-ready structure
- Clear prioritization logic
