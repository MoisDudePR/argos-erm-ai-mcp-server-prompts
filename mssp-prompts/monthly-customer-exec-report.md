# Monthly Customer Executive Report

## MSSP Use Case
Deliver consistent executive reporting to customers.

## MCP Context
- Risk trends
- Key exposures
- Remediation progress

## Prompt
You are preparing a monthly executive report for a managed customer.

Using ERM MCP data:
- Summarize current external risk posture
- Highlight notable changes since last month
- Show remediation progress and remaining risk

Provide:
1. Executive summary
2. Top risks
3. Progress highlights
4. Next-month focus areas

## Expected Output
- Polished executive tone
- Customer-ready format
