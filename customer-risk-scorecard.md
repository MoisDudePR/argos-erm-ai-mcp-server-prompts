# Customer Risk Scorecard

## MSSP Use Case
Generate a standardized external risk scorecard for a single customer.

## MCP Context
- Customer-specific assets
- Exposure severity
- Risk trends
- Threat intelligence

## Prompt
You are preparing a recurring customer risk scorecard.

Using ERM MCP data for the selected tenant:
- Summarize overall external risk posture
- Highlight changes since the last reporting period
- Identify top contributors to risk

Provide:
1. Overall risk summary
2. Key exposure drivers
3. Trend analysis
4. Recommended actions

## Expected Output
- Repeatable format
- Customer-facing language
