# Risk Trend Explanation

## CSM Use Case
Explain risk changes to customers.

## MCP Context
- Historical exposure data
- Risk scoring trends

## Prompt
A customer asks why their external risk score changed.

Using ERM MCP data:
- Explain the drivers behind risk score changes
- Clarify whether changes are positive or negative
- Recommend follow-up actions

Provide:
1. Clear explanation
2. Key contributing factors
3. Next steps

## Expected Output
- Reassuring tone
- Easy-to-understand explanation
