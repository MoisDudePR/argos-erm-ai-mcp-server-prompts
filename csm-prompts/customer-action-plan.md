# Customer Action Plan

## CSM Use Case
Create a clear, agreed-upon next-steps plan.

## MCP Context
- Outstanding exposures
- Customer priorities
- Risk trends

## Prompt
You are creating a customer action plan following a review meeting.

Using ERM MCP data:
- Identify priority actions for the customer
- Align actions to risk reduction
- Set expectations for the next review period

Provide:
1. Action plan summary
2. Priority actions
3. Success criteria

## Expected Output
- Collaborative tone
- Clear accountability
