# Alert Triage Assistance

## Use Case
Support rapid triage of ERM alerts.

## MCP Context
- Alerts
- Asset criticality
- Exposure data
- Threat intelligence

## Prompt
You are assisting a SOC analyst with ERM alert triage.

For each alert:
- Explain what triggered the alert
- Assess severity and likelihood
- Identify affected assets
- Recommend next investigative steps

Provide:
1. Alert summary
2. Risk assessment
3. Triage decision guidance

## Expected Output
- Clear, concise analysis
- SOC-friendly language
- Action-oriented guidance
