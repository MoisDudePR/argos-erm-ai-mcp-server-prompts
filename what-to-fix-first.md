# What Should We Fix First?

## Demo Goal
Demonstrate ERM’s prioritization capabilities.

## MCP Context
- Exposure severity
- Exploit intelligence
- Asset criticality

## Prompt
A customer asks: “If we do nothing else, what should we fix first?”

Using ERM MCP data:
- Identify the highest-priority remediation action
- Explain why it matters most
- Keep the explanation simple

Provide:
1. Top remediation priority
2. Risk-based justification
3. Expected risk reduction

## Expected Output
- Decisive guidance
- Clear prioritization logic
