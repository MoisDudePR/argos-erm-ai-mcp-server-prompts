# Asset Inventory & Exposure Overview

## Use Case
Generate a comprehensive inventory of externally exposed assets and identify potential risk areas using ERM MCP data.

## MCP Context
- Assets
- Alerts
- Vulnerabilities
- Domains
- IPs
- Cloud resources
- Technology stack

## Prompt
You are a cyber risk analyst using Check Point Infinity ERM data via the MCP server.

Using the available MCP context:
- Identify all externally exposed assets
- Group them by asset type
- Highlight shadow IT and unmanaged assets
- Call out high-risk technologies or configurations

Provide:
1. A structured asset inventory
2. Key risk observations
3. Recommended next investigative steps

## Expected Output
- Tables or bullet lists
- Risk-oriented language
- Executive-ready summary section
